1.1 Sejarah HTML

HTML adalah bahasa dasar untuk menampilkan halaman web pada web browser . HTML merupakan singkatan dari Hypertext Markup Language. Yang artinya adalah bahasa markup(penanda) berbasis text atau bisa disebut sebagai formatting language. HTML adalah bahasa pengrograman. HTML bukanlah bahasa pemprograman teteapi bahasa markup formatting.

HTMLdibuat oleh Tim Berners-Lee ketika masih bekerja untuk CERN dan dipopulerkan pertama kali oleh browser Mosaic. Selama awal tahun 1990 HTML mengalami perkembangan yang sangat pesat. Setiap pengembangan HTML pasti akan menambahkan kemampuan dan fasilitas yang lebih baik dari versi sebelumnnya. Sebelum suatu HTML diadakan sebagai suatu dokumen HTML standar, ia baru disetujui dulu oleh W3C untuk dievaluasi secara ketat. Setiap terjadi perkembangan suatu versi HTML, maka mau tak mau browser pun harus memperbaiki diri agar bisa mendukung kode-kode HTML yang baru tersebut. Sebab jika tidak, broswer tak akan bisa menampilkan HTML tersebut.

Saat ini HTML sudah mencapai versi 5 atau biasa disebut HTML 5. Versi ini memiliki banyak fitur yang dapat digunakan dan memiliki beberapa pengguna script pada editor.

HTML biasa ditulis pada sebuah text edit (dreamweaver,notepad++, notepad, phpstrm, bracker, netbeans,dll). Ketik kutipan coding HTML diatas pada texteditor anda dan kemudian simpan dengan format extention .html yang menandakan bahwa file tersebut adalah file HTML.

1.2 Perlengkapan Pendukung Belajar HTML

Item yang paling pending dalam melakukan pemprograman web adalah editor teks atau IDE yang mampu mendukung Anda dalam menulis kode. Sejumlah alat pendukung yang dapat dipilih mulai dari Windows Notepad yang memungkinkan Anda menulis kode ke file sampai Adobe Dreamweaver yang membuat kode web dalam lingkungan WYSUWYG (What You See Is What You Get).
